My Projects

Parking management System

The "Parking Management System" is a software application designed to efficiently manage parking spaces in a facility. Developed using Java for the backend logic and SQL for database management, the system allows users to register vehicles, assign parking spots, and track the availability of parking spaces in real-time. It includes features like user authentication, parking slot reservation, and detailed reporting. This system helps streamline the parking process, reduce manual errors, and optimize the usage of parking resources in various environments such as malls, offices, or residential complexes.

skillBridge - freelancing website

The freelancing website developed using the MERN stack (MongoDB, Express.js, React.js, and Node.js) is a platform that connects freelancers with clients seeking various services. The site allows freelancers to create profiles, showcase their skills, and bid on projects posted by clients. Clients can post job requirements, review freelancer profiles, and hire the best candidates for their projects. With a modern user interface built in React, a robust backend using Node.js and Express, and efficient data management with MongoDB, this platform ensures a seamless and interactive experience for both freelancers and clients
